# Doctor Appointment Booking System

## Overview
A full-stack MERN Doctor Appointment Booking System that enables users to book appointments online, doctors to manage their schedules, and administrators to manage doctors and appointments through a dedicated dashboard.

## Features
- User authentication (JWT)
- Doctor authentication
- Admin dashboard
- Book appointments
- Cancel appointments
- Doctor profile management
- Cloudinary image upload
- Razorpay payment integration
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Tailwind CSS
- JWT Authentication
- Cloudinary
- Razorpay
- Render
- Vercel

## Project Structure
Doctor-Appointment-Booking-System
├── frontend
├── backend
└── admin

## Installation
1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Run frontend, backend, and admin
